Quick Links

Next Steps

  1. Make sure your .env file is configured with Discord credentials
  2. Register slash commands: php register_commands.php
  3. Expose this endpoint with ngrok: ngrok http 9999
  4. Set the ngrok HTTPS URL in Discord Developer Portal
  5. Set up cron job to process messages every minute

Available Commands

  • /checkpoint - Start or close a receipt tracking checkpoint
  • /undo - Undo the latest checkpoint
